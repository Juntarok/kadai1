# 実行手順
１rasberry piの16,18,22,30,34,39のピンにジャンパー線をさす

２コマンドを実行する


# インストール
    $ git clone https://github.com/Juntarok/kadai1.git
    $ cd kadai1
    $ make
    $ sudo insmod myled.ko
    $ sudo chmod 666 /dev/myled0

# LEDを点灯
    $ echo 0 >/dev/myled0
上記を入力するとLED1,LED2,LED3が順番に点滅する

# LEDを消灯
    $ echo 2 >/dev/myled0
上記を入力するとLED1,LED2,LED3が消灯する

# アンインストール
    $ sudo rmmod myled
